apply()#

StateInputOperator.apply(state, input_)[source]#

Apply the operator to the given state / input: \(\Ophat_{\ell}(\qhat,\u) = \Nhat[\u\otimes\qhat]\).

Parameters:
state(r,) ndarray

State vector.

input_(m,) ndarray

Input vector.

Returns:
out(r,) ndarray

The evaluation \(\Nhat[\u\otimes\qhat]\).
